### 2. File database `workspace_manager.db` đã bị track / nằm ngoài gitignore

**File:** `backend/workspace_manager.db` (40KB)

**Triệu chứng:** File database SQLite đang nằm ngay trong thư mục `backend/` và CHƯA được gitignore.

**Nguy hiểm:** Database này chứa data thực. Nếu push lên Git, bất kỳ ai cũng đọc được. Và mỗi lần db thay đổi, Git sẽ coi là thay đổi → commit history rẻ.

**Cách sửa:**
1. Thêm `*.db` vào `.gitignore`
2. Nếu đã commit rồi: `git rm --cached backend/workspace_manager.db`
3. Nên dùng biến môi trường cho DATABASE_URL thay vì hardcode

### 3. DATABASE_URL hardcode trực tiếp trong code

**File:** `backend/app/db.py` (dòng 6)

```python
DATABASE_URL = "sqlite:///./workspace_manager.db"
```

**Triệu chứng:** Database URL được viết thẳng vào code thay vì đọc từ biến môi trường (`.env`).

**Nguy hiểm:**
- Khi deploy lên production, anh sẽ phải SỬA CODE để đổi database → rủi ro commit nhầm
- Không thể chạy nhiều môi trường (dev / staging / production) với cùng codebase
- Theo plan, production sẽ dùng PostgreSQL → hiện tại không có cách nào switch

**Cách sửa:**
```python
import os
DATABASE_URL = os.getenv("DATABASE_URL", "sqlite:///./workspace_manager.db")
```

### 4. Không có Authentication/Authorization cho API

**Files:** Tất cả routers (`workspaces.py`, `members.py`, `invites.py`)

**Triệu chứng:** Tất cả 7 endpoints đều PUBLIC, không yêu cầu đăng nhập hay token.

**Nguy hiểm:** Bất kỳ ai biết URL backend đều có thể:
- Xem danh sách workspace và members (lộ email)
- Kick bất kỳ member nào
- Gửi invite spam
- Cancel invite của người khác

**Đây là vấn đề NGHIÊM TRỌNG NHẤT** vì plan.md section 12 đã liệt kê rõ: "Admin login, API rate limits, Token encryption"

**Cách sửa (v1 minimum):**
```python
# backend/app/main.py - thêm middleware auth đơn giản
from fastapi import Depends, HTTPException, Header

async def verify_admin_token(authorization: str = Header(...)):
    expected = os.getenv("ADMIN_TOKEN")
    if not expected or authorization != f"Bearer {expected}":
        raise HTTPException(status_code=401, detail="Unauthorized")
```

## 🟡 Warnings (Nên sửa)

### 1. `datetime.utcnow()` đã bị deprecated

**Files:** `backend/app/models.py` (dòng 18, 41), `backend/app/routers/members.py` (dòng 27), `backend/tests/conftest.py` (dòng 31, 39, 46)

**Triệu chứng:** Dùng `datetime.utcnow()` → Python 3.12+ cảnh báo deprecated.

**Nếu không sửa:** Khi nâng Python version, sẽ hiện warning liên tục. Code có thể lỗi ở Python tương lai.

**Cách sửa:** Thay bằng `datetime.now(timezone.utc)`:
```python
from datetime import datetime, timezone
# thay datetime.utcnow() → datetime.now(timezone.utc)
```

---

### 2. `@app.on_event("startup")` đã deprecated

**File:** `backend/app/main.py` (dòng 12)

**Triệu chứng:** FastAPI không khuyến khích dùng `on_event` nữa, khuyên dùng `lifespan` thay thế.

**Cách sửa:**
```python
from contextlib import asynccontextmanager

@asynccontextmanager
async def lifespan(app: FastAPI):
    init_db()
    yield

app = FastAPI(title="Workspace Manager API", lifespan=lifespan)
```

### 3. Endpoints nhận `dict` thay vì Pydantic model

**Files:** `members.py` (dòng 15, 35), `invites.py` (dòng 28, 42)

**Triệu chứng:** Các endpoint `POST`/`DELETE` nhận `payload: dict` thay vì dùng Pydantic schema đã tạo sẵn trong `schemas.py`.

**Nguy hiểm:**
- Không có validation tự động → client gửi data sai → lỗi tiềm ẩn
- Không có API docs đẹp (Swagger/OpenAPI sẽ không hiện fields rõ ràng)
- File `schemas.py` đã tạo nhưng KHÔNG ĐƯỢC DÙNG → dead code

**Cách sửa:** Tạo Pydantic request models:
```python
class InviteRequest(BaseModel):
    org_id: str
    email: str
    role: str = "member"

@router.post("/api/invite")
def invite_member(payload: InviteRequest, ...):
```

## 🟢 Suggestions (Tùy chọn - Nên làm khi có thời gian)

### 1. Thêm CORS middleware cho backend

Backend FastAPI chưa config CORS. Khi frontend chạy riêng port, sẽ bị browser chặn.

```python
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:3000"],
    allow_methods=["*"],
    allow_headers=["*"],
)
```

### 2. Thêm Invite type vào `frontend/src/types/api.ts`

Chỉ có `Workspace` và `Member` types, thiếu `Invite` type.

- Giữ nguyên (audit trail) - OK nếu có filter trên frontend
